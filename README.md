# Test-GraphQL-App

>The demo for practicing GraphQL.
>
>Node.js + Koa + MongoDB + GraphQL + Vue.
>
>Article link: https://www.cnblogs.com/wisewrong/



## 一、目录介绍

目录中带有 `-old` 后缀的文件，是指使用传统 API 格式的文件，在运行 GraphQL 服务的时候不会使用。

### 1. server - 后端服务

启动服务

```
npm run start
```

### 2. fe - 前端应用

启动应用

```
npm run dev
```
